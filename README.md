# msgAdmin

A MQTT message manager tool.

## MQTT Broker setting

Host : localhost
Port : 1883

## Quick Start

1. cd msgAdmin-master
2. npm install
3. index <http://localhost:3000>
4. node-red <http://localhost:3000/red>
